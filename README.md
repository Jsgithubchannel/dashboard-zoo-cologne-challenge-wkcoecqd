## Tech stack
- TypeScript project using nuxt3 + vue3
- Styling: TailwindCSS
- Testing: Vitest
- Linting: ESLint

## Getting Started

### Prerequisites
- Node.js
- pnpm

### Installiation
```bash
    pnpm install
```

### Development
- Start the development server:
```bash
    pnpm run server
```
Open http://localhost:3000 to view the dashboard.

### Testing
```bash
    pnpm run test
```

### Linting
```bash
    pnpm run lint
```
